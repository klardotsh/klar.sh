# klar.sh

This is the website hosted at the root of `http(s)?://(www.)?klar.sh`.

## Generating The Site

- Ensure a POSIX-compliant `/bin/sh` and `make` are installed, as well as
  [lowdown](https://kristaps.bsd.lv/lowdown/), a Markdown parser.
- Just run `make`.
